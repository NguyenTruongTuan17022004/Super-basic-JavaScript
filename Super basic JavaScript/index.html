Comparison Table of var, let, const in JavaScript

Feature var let const
Scope (Where the variable can be used?) Available throughout the entire function (if declared inside a function) Available only inside the block `{}` where it was declared Available only inside the block `{}` where it was declared
Can the value be changed after declaration? ✅ Can be changed ✅ Can be changed ❌ Cannot be changed
Can it be redeclared with the same name? ✅ Can be redeclared ❌ Cannot be redeclared ❌ Cannot be redeclared

P.S: If you assign a value to a variable without using var, let, or const, it will implicitly become a global variable.
